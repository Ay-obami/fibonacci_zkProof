# Fibonacci ZK Circuit (Circom + PLONK)

A Circom circuit proving knowledge of two private starting values of a
Fibonacci-like sequence, without revealing them — built with PLONK proofs
via snarkjs, with an exported Solidity verifier for on-chain verification.

## Prerequisites

- [circom](https://docs.circom.io/getting-started/installation/) v2+
- [snarkjs](https://github.com/iden3/snarkjs) (`npm install -g snarkjs`)
- Node.js

## Project structure

```
src/
├── fibonacci.circom      # circuit source
├── input.json             # sample private/public inputs
├── run_plonk.sh            # compile → prove → verify → export verifier
contracts/
└── fibonacciVerifier.sol   # generated Solidity verifier (deployable)
```

## Usage

Compile the circuit, generate a witness, run the PLONK trusted setup,
generate a proof, verify it, and export a Solidity verifier — all in
one command:

```bash
chmod +x run_plonk.sh
./run_plonk.sh fibonacci input.json 12
```

Arguments: `<circuit_name> <input_file> [ptau_power]` (default power: 12,
i.e. 2^12 constraints — raise it if your circuit is larger).

Generated artifacts land in `build/` (gitignored — regenerate anytime
by re-running the script). Powers of Tau files are cached in `ptau/`
after the first run so subsequent runs skip that step.

## Deploying the verifier

Copy `contracts/fibonacciVerifier.sol` into your Foundry or Hardhat
project's `src/`/`contracts/` folder, deploy it, and call `verifyProof()`
with the `proof.json` / `public.json` values generated by the script.
